# Project Stage Gates

Sprint 8.2 adds a lightweight PRINCE2-style stage-gate model.

## Delivery Stages

Default stages:

1. Initiation
2. Planning
3. Delivery
4. Validation
5. Handover
6. Closure

## Gate Status

Each stage gate can be:

- Not started.
- In progress.
- Complete.
- Blocked.

## Lightweight Criteria

Criteria remain advisory in Sprint 8.2:

- Initiation: Kick-off milestone complete.
- Planning: Resources scheduled and equipment order task complete.
- Delivery: Deployment commenced task complete.
- Validation: UAT milestone complete where applicable.
- Handover: Support handover form exists.
- Closure: Closure form exists and project status is Closed.

Stage gates are visible on the project overview and included in weekly update exports.
